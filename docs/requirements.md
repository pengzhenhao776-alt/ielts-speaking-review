# 产品需求文档

## 项目概述
雅思口语学习卡片软件，供雅思口语老师制作学习卡片并分享给学生复习使用。

## 用户角色
- **老师**：创建、管理、分享卡片组和口语思路模板
- **学生**：通过分享链接查看卡片和模板，无需登录

## 功能需求

### 模块一：表达卡片
- 创建卡片组（如"健康话题词汇"、"Part1高频表达"）
- 卡片内容：正面（表达/词汇）、背面（中文释义 + 英文例句 + 场景标签 Part1/Part2/Part3）
- 翻转卡片交互（点击翻转查看背面）
- 卡片组内翻页浏览
- 一键生成分享链接

### 模块二：口语思路模板
- 按话题创建思路模板
- 三段式框架：开头 / 主体 / 结尾
- 每段可添加关键要点
- 生成分享链接

### 模块三：学生复习端
- 通过分享链接进入，无需登录
- 只读模式查看卡片组和模板
- 卡片翻转 + 左右滑动翻页
- 模板展开/折叠浏览

## 分发方式
- 老师制作内容 → 生成分享链接 → 复制发送给学生
- 学生打开链接即用，无需安装

## 数据存储
- 老师端：浏览器 localStorage 持久化
- 学生端：仅从 URL 读取，不写入本地

## 非功能需求
- 响应式设计，适配手机/平板/电脑
- Apple 风格简洁 UI
- 无需后端服务，纯前端静态站点
